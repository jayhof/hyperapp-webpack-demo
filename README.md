# hyperapp-webpack-demo

A quick demonstration on how to set up [hyperapp](https://github.com/hyperapp/hyperapp) with [webpack 4](https://github.com/webpack/webpack).

## Installation

### Prerequisites
- node.js ^8.0.0
- yarn ^1.0.0

### Setup
```sh
$ yarn install
```

## Development
```sh
$ yarn start:dev
```
